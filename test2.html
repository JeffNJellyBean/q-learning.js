This is a practical implementation of the q-learning algorithm to learning in a game. The goal is to catch the green (+1 point) while avoiding the gray (-1 point). Both types are added to the game with equal distribution. The actions are move left, or move right, or stay. An exploration/exploitation ratio is 0.1. With this probability a random action is taken.

With no learning the score should make a random walk around 0. With learning the score should increase.
